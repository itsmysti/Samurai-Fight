# Samurai Fight!
> A simple fight game made with Javascript!


## Controls for player 1

```sh
  A - Moves backwards
  W - Jump
  D - Moves forward  
  SpaceBar - Attack
```

## Controls for player 2

```sh
  ArrowsLeft - Moves forward
  ArrowsUp - Jump 
  ArrowsRight - Moves backwards  
  ArrowDown - Attack
```
